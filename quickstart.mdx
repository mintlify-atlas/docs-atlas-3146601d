Get your first AI-powered commit done in minutes with YitPush.

<Steps>
  <Step title="Install YitPush">
    Install YitPush globally using the .NET CLI:

    ```bash
    dotnet tool install --global YitPush
    ```

    <Tip>
      To uninstall later: `dotnet tool uninstall --global YitPush`
    </Tip>
  </Step>

  <Step title="Get your DeepSeek API key">
    1. Sign up at [DeepSeek Platform](https://platform.deepseek.com/)
    2. Navigate to **API Keys** and create a new key
    3. Copy your API key for the next step
  </Step>

  <Step title="Set your API key">
    Configure the `DEEPSEEK_API_KEY` environment variable:

    <Tabs>
      <Tab title="Linux/macOS">
        ```bash
        export DEEPSEEK_API_KEY='your-api-key-here'
        
        # Make it permanent
        echo 'export DEEPSEEK_API_KEY="your-api-key-here"' >> ~/.zshrc
        ```
      </Tab>
      <Tab title="Windows PowerShell">
        ```powershell
        $env:DEEPSEEK_API_KEY='your-api-key-here'
        
        # Make it permanent (run as Admin)
        [System.Environment]::SetEnvironmentVariable('DEEPSEEK_API_KEY', 'your-api-key-here', 'User')
        ```
      </Tab>
    </Tabs>

    <Warning>
      Restart your terminal after setting the environment variable for it to take effect.
    </Warning>
  </Step>

  <Step title="Make your first commit">
    Navigate to any git repository and run:

    ```bash
    yitpush commit
    ```

    YitPush will:
    - ✅ Analyze your staged, unstaged and untracked changes
    - ✅ Generate a smart conventional commit message with DeepSeek AI
    - ✅ Run `git add .`, `git commit` and `git push` automatically
    - ✅ Copy the commit message to your clipboard

    Example output:
    ```
    🚀 YitPush - AI-Powered Git Commit Tool

    📊 Analyzing git changes...
    Found changes (1234 characters)

    🤖 Generating commit message with DeepSeek Reasoning...

    📝 Generated commit message:
       "feat: add user authentication with JWT tokens"

    ⏩ Proceeding automatically (use --confirm to review)...

    ⚙️  Executing git commands...
       git add .
       git commit -m "feat: add user authentication with JWT tokens"
       git push

    ✅ Successfully committed and pushed changes!
    ```
  </Step>
</Steps>

## Common issues

<AccordionGroup>
  <Accordion title='"DEEPSEEK_API_KEY not found"'>
    Make sure you've set the environment variable and **restarted your terminal**.
    
    Verify it's set:
    ```bash
    echo $DEEPSEEK_API_KEY  # Linux/macOS
    echo $env:DEEPSEEK_API_KEY  # Windows PowerShell
    ```
  </Accordion>

  <Accordion title='"Not a git repository"'>
    Make sure you're running `yitpush` from inside a git repository.
    
    Initialize one if needed:
    ```bash
    git init
    git remote add origin <your-repo-url>
    ```
  </Accordion>

  <Accordion title='"git push failed"'>
    Verify you have:
    - A remote configured: `git remote -v`
    - Valid credentials for the remote
    - Network connectivity
    
    The tool will prompt you to set upstream if your branch doesn't have one.
  </Accordion>
</AccordionGroup>
