# Introduction

YitPush is an AI-powered Git commit tool that automates your entire git workflow with a single command. Using DeepSeek's reasoning AI, it analyzes your changes, generates meaningful commit messages, and pushes everything automatically.

```bash
yitpush commit
```

That's it. YitPush handles staging, committing with an intelligent message, and pushing to your remote repository.

## Why use YitPush?

YitPush replaces the repetitive `git add . && git commit -m "..." && git push` workflow with intelligent automation that understands your code changes.

## Key features

### Smart commit messages

YitPush analyzes your staged, unstaged, and untracked changes to generate conventional commit messages that accurately describe what changed and why.

### Flexible workflows

- **Automatic mode**: Commit and push immediately
- **Confirmation mode**: Review the AI-generated message before committing with `--confirm`
- **Detailed mode**: Generate commits with title, body, and bullet points using `--detailed`
- **Multilingual**: Generate messages in any language with `--language`

### Beyond commits

YitPush isn't just for commits:
- **Branch checkout**: Interactive branch switching with date sorting
- **PR descriptions**: Generate comprehensive pull request descriptions
- **Azure DevOps**: Manage repositories and variable groups
